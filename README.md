# js-snippets (wip)

Basic JS snippets.

Also an attempt to share JS snippets with VS code and vim.

### Globals

|  Trigger | Content       |
| -------: | ------------- |
|  `cl→`   | console.log   |
|  `f→`    | arrow func    |
|  `b→`    | braces        |


## Settings

The `editor.snippetSuggestions` setting in vscode `settings.json` will show snippets on top of the suggestion list.

```json
"editor.snippetSuggestions": "top"
```
